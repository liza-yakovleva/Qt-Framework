# Changelog

Всі видатні зміни у цьому проекті документуються в цьому файлі.

## [1.0.0] - 2026-03-08

### Dodano (Added)
- ✨ Основна функціональність форми опитування
- 🎨 High-contrast UI з кастомними QSS стилями
- 📝 Валідація даних користувача (12 валідаційних правил)
- 💾 Безпечне збереження результатів у файл з UTF-8 кодуванням
- 📊 Часові мітки для кожного запису опитування
- 🏗️ Модульна архітектура (7 модулів)

### Структура модулів
- `main.py` - Точка входу та головне вікно (81 рядок)
- `config.py` - Конфігурація та константи (30 параметрів)
- `styles.py` - QSS стилізування (68 рядків CSS-подібного коду)
- `models.py` - Структури даних SurveyData з dataclass
- `ui.py` - Побудова інтерфейсу через UIBuilder (137 рядків)
- `handlers.py` - Обробка подій SurveyEventHandler (105 рядків)
- `utils.py` - Утилітарні функції (65 рядків)

### Технічні особливості
- ✅ SOLID принципи проектування
- ✅ DRY (Don't Repeat Yourself)
- ✅ Розділення забезпечення (Separation of Concerns)
- ✅ Документація через docstrings
- ✅ Type hints для Python функцій
- ✅ Обробка помилок та винятків

### Стрес-тестування
- Успішне введення 50+ записів у файл
- Тестування на максимальних значеннях (вік 100, текст 10000+ символів)
- Перевірка UTF-8 символів (кирилиця, спеціальні символи)
- Валідація обробки файлових помилок (нема прав на запис)
