# Survey Pro  | Python Qt Application

**Курс:** Комп'ютерна інженерія  
**Дисципліна:** Розробка інтерфейсів користувача 
**Виконала:** Яковлева Єлизавета Дмитрівна  

##  Огляд проєкту
Даний застосунок є частиною циклу лабораторних робіт. У цій ітерації реалізовано форму опитування за допомогою **Python 3** та **PySide6** (офіційна реалізація Qt для Python).



##  Технічні особливості
- **High-Contrast UI:** Використано кастомні стилі **QSS** для забезпечення максимальної читабельності тексту незалежно від системної теми.
- **Layout Management:** Інтерфейс побудований на базі `QVBoxLayout` та `QHBoxLayout`, що забезпечує адаптивність елементів при зміні розмірів вікна.
- **Data Validation:** Реалізовано перевірку обов'язкових полів введення перед фіналізацією запису.
- **Safe I/O:** Збереження даних виконується через контекстні менеджери (`with open`) у кодуванні UTF-8 для запобігання втрати даних та витоків пам'яті.

##  Архітектура
Проєкт реалізовано в рамках об'єктно-орієнтованої парадигми (ООП). Весь графічний інтерфейс описаний декларативно у коді, що є сучасним стандартом для гнучкої розробки динамічних GUI.



##  Структура файлів
- `main.py` — головний файл застосунку з логікою та UI.
- `survey_results.txt` — вихідний файл із результатами (генерується автоматично).
- `.gitignore` — конфігурація для чистоти репозиторію.
